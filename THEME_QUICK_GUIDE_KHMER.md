# 🎨 ការណែនាំអំពី Theme - ជំនាន់ខ្លី

## Themes ដែលមាន

កម្មវិធីឥឡូវនេះមាន **5 themes** ដែលអ្នកអាចជ្រើសរើសបាន៖

### 1. ☀️ Default (ដើម) - ពន្លឺ
- **រចនាប័ទ្ម**: ពណ៌សភ្លឺ មើលងាយ
- **ល្អសម្រាប់**: ការិយាល័យ ពេលថ្ងៃ
- **ពណ៌**: ផ្ទៃស ប៊្លូ #1890ff

### 2. 🌑 Black (ខ្មៅ) - ងងឹត
- **រចនាប័ទ្ម**: ពណ៌ខ្មៅ ការពារភ្នែក
- **ល្អសម្រាប់**: ពេលយប់ មើលបានយូរ
- **ពណ៌**: ផ្ទៃខ្មៅ ខៀវចម្រុះ #00d4ff

### 3. 💜 Purple (ស្វាយ) - ស្រស់ស្អាត
- **រចនាប័ទ្ម**: ពណ៌ស្វាយ gradient ស្អាត
- **ល្អសម្រាប់**: ការងារច្នៃប្រឌិត
- **ពណ៌**: ស្វាយ #9333ea ផ្ទៃ lavender

### 4. 💙 Blue (ខៀវ) - វិជ្ជាជីវៈ
- **រចនាប័ទ្ម**: ប៊្លូ gradient វិជ្ជាជីវៈ
- **ល្អសម្រាប់**: អាជីវកម្ម ការិយាល័យ
- **ពណ៌**: ប៊្លូ #2196f3 ផ្ទៃ sky blue

### 5. 💚 Green (បៃតង) - ធម្មជាតិ
- **រចនាប័ទ្ម**: បៃតងស្រស់ មើលងាយ
- **ល្អសម្រាប់**: ការងារបានយូរ មិនចង្អៀត
- **ពណ៌**: បៃតង #52b788 ផ្ទៃ mint

---

## 📍 របៀបផ្លាស់ប្តូរ Theme

### ជំហានទី 1: បើក Settings
- ចុចលើ tab **"⚙ ការកំណត់ (Settings)"**

### ជំហានទី 2: ចូល Configuration
- នៅក្នុងម៉ឺនុយខាងឆ្វេង ចុច **"⚙ Configuration"**

### ជំហានទី 3: ជ្រើសរើស Theme
- នៅខាងលើ នឹងឃើញផ្នែក **"🎨 Theme Selection"**
- ចុចលើ dropdown menu
- ជ្រើសរើស theme ដែលអ្នកចូលចិត្ត

### ជំហានទី 4: អនុវត្ត Theme
- ចុចប៊ូតុង **"✨ Apply Theme"**
- Theme នឹងផ្លាស់ប្តូរភ្លាមៗ!
- ការកំណត់ត្រូវបានរក្សាទុកស្វ័យប្រវត្តិ

### ជំហានទី 5: រីករាយ!
- Theme ថ្មីត្រូវបានអនុវត្តពេញលេញ
- ពេលបើកកម្មវិធីម្តងទៀត theme នេះនឹងនៅដដែល

---

## 💾 ការរក្សាទុក Theme

### ស្វ័យប្រវត្តិ 100%
- ពេលអ្នកជ្រើសរើស theme theme ត្រូវបានរក្សាទុក
- ឯកសារ: `app_settings.json`
- ពេលបើកកម្មវិធីម្តងទៀត theme ត្រូវបាន load ដោយស្វ័យប្រវត្តិ

### គំរូ Settings
```json
{
    "selected_theme": "Purple"
}
```

## ✨ លក្ខណៈពិសេស

### 1. **ផ្លាស់ប្តូរភ្លាមៗ**
- មិនបាច់បិទកម្មវិធី
- ចុច "Apply Theme" ហើយឃើញភ្លាមៗ

### 2. **រក្សាទុកស្វ័យប្រវត្តិ**
- Theme របស់អ្នកត្រូវបានរក្សាទុក
- បើកម្តងទៀតនៅដដែល

### 3. **រចនាវិជ្ជាជីវៈ**
- Themes ទាំង 5 ត្រូវបានរចនាយ៉ាងស្អាត
- ពណ៌ស្រស់ស្អាត មើលងាយ

### 4. **គ្របដណ្តប់ពេញលេញ**
- Themes ប៉ះពាល់គ្រប់ UI elements:
  - ✅ Buttons (ប៊ូតុង)
  - ✅ Input fields (កន្លែងបញ្ចូល)
  - ✅ Tables (តារាង)
  - ✅ Tabs (Tabs)
  - ✅ Progress bars (របារវឌ្ឍនភាព)
  - ✅ Scrollbars (របាររមូរ)
  - ✅ Backgrounds (ផ្ទៃ)
  - ✅ Text colors (ពណ៌អក្សរ)

## 🎯 ការណែនាំរហ័ស

### ចង់បាន Theme ណា?

**ពេលថ្ងៃ / ការិយាល័យ:**
→ ប្រើ **Default** ☀️ ឬ **Blue** 💙

**ពេលយប់ / មើលបានយូរ:**
→ ប្រើ **Black** 🌑 ឬ **Green** 💚

**ចង់បានស្អាត / ច្នៃប្រឌិត:**
→ ប្រើ **Purple** 💜

**មិនច្បាស់ / ចង់សាកល្បង:**
→ ចាប់ផ្តើមជាមួយ **Default** ☀️ មុន

---

## 🔄 របៀបត្រឡប់ទៅ Theme ដើម

ប្រសិនបើអ្នកចង់បាន theme ដើមវិញ:

1. ចូល Settings → Configuration
2. ជ្រើសរើស **"☀️  Default (Light)"**
3. ចុច **"✨ Apply Theme"**
4. រួចហើយ!

---

- **រក្សាទុកស្វ័យប្រវត្តិ** - មិនបាច់ចុច Save ទេ
